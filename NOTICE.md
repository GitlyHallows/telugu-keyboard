# Notices

Telugu Keyboard is an independent macOS input method.

Design references used while building the project:

- Unicode Telugu script documentation for code point and abugida composition behavior.
- Public behavior of ranked Telugu transliteration tools, used only as development benchmarks.
- LipikaIME was inspected as an InputMethodKit packaging reference, but GPL source code is not copied into this project.

The shipped keyboard does not include third-party transliteration service code and does not call network transliteration services at runtime.
